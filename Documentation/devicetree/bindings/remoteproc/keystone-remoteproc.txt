* Texas Instruments Keystone remoteproc driver

Required properties:
 - compatible :	Should be "ti,keystone-rproc".
 - reg :	This property defines the address and length of the
		registers used to reset, start and interrupt a DSP core.
 - addr-map:	This property defines the memory address map used by the rproc device
		to download and boot a DSP firmware image.
		Each tuple consists of a DSP core's
		global address, local address and size.
 - core :	Name of this remote processor device.
 - firmware :	Name of DSP firmware image to load.
 - core-dump-mask: Mask which is used to identify which DSP core has crashed.

Example:
rproc0: rproc@2620040 {
			compatible = "ti,keystone-rproc";
			reg = <0x02620040 4				/* boot address register */
			       0x02350A5C 4				/* module control register */
			       0x02350120 4				/* power domain transition command register */
			       0x02620240 4				/* ipc interrupt generation register */
			       0x026202BC 4>;				/* ipc arch register */
			addr-map = <0x10800000 0x00800000 0x100000	/* l2-ram  global, local, size */
				    0x0c000000 0x0c000000 0x200000	/* msmc-sram  global, local, size */
				    0xa0000000 0xa0000000 0x20000000>;	/* ddr3-sram  global, local, size */
			core = "dsp-core0";
			firmware = "dsp-core0.out";
			core-dump-mask	= <0x00000010>;			/* ipc arch register mask */
		};
